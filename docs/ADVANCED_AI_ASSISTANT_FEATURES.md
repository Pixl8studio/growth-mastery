```tsx

  <div className="h-full bg-white" style={{ width: `${(currentStep / 12) * 100}%` }} />
</div>
```

Shows:

- Current step number
- Total steps (12)
- Percentage complete
- Visual progress bar

Pre-configured prompts that users can click:

```tsx

  { label: "Fill Form", prompt: "Help me fill in this form" },
```

Benefits:

- **Faster interaction** - No typing needed
